01 · The Problem

Individual Models Have a Fatal Trade-off

The Abstention Problem
27.2%

The Accuracy Ceiling
78.0%
Gemini's best all-500 accuracy. Models that answer everything sacrifice accuracy. No individual model wins on both dimensions simultaneously.
No Internal Error Check
A solo model cannot review its own reasoning. It is confidently wrong with no recourse — and structurally has no way to detect it.
The Council Solution
Force heterogeneous models to cross-examine each other before committing. Disagreement becomes the signal. Debate forces re-examination.
02 · Project Context

What This Project Is Actually About

"Can structured disagreement between AI models produce more reliable decisions than any single model alone?"

Large language models have reached human-level performance on medical benchmarks. But benchmark accuracy is not deployment accuracy. Models fail silently, abstain unpredictably, and have no mechanism to catch their own errors.

This project applies the LLM Council architecture — inspired by Andrej Karpathy's multi-model deliberation framework — to structured medical QA classification. Rather than routing a question to one model, we route it to three heterogeneous models, force them to debate, and let disagreement surface errors before they reach a verdict.

The domain is medical question answering: a setting where overconfident wrong answers have real consequences, where multiple valid reasoning paths exist, and where the gap between benchmark performance and real-world robustness is most consequential.

04 · The Data

Dataset Design and Stratification

Source Dataset
sampled_batch.jsonl
500 four-option multiple-choice questions from a medical QA corpus. Each question has a single gold-standard correct answer. The format mirrors USMLE-style questions — the same format used by MedQA and MedFuzz — making results directly comparable to published benchmarks.
Stratification
Stratified along domain and difficulty axes. Proportional allocation within each stratum ensures no single category dominates the accuracy estimate.

Why Stratification Matters
The n = 100 pilot was unstratified. If cardiology questions clustered in that sample, or if easy questions were overrepresented, the accuracy estimate is unreliable regardless of sample size. Stratification ensures the council's accuracy reflects genuine breadth — not a lucky draw of questions skewed toward its strengths.
Confidence Threshold
Each agent operates with τ = 0.60. Answers below this threshold produce an abstention rather than a guess. A parallel solo run at τ = 0.0 (no abstention) was also conducted, providing the uncensored baseline for apples-to-apples comparison with the council's all-500 accuracy.
05 · Debate Mechanics

Why Disagreement Is the Signal, Not the Noise

01
Error Cancellation
Heterogeneous models make uncorrelated errors
Claude, Gemini, and o3 have structurally different failure modes. A question that trips Claude's pattern-matching does not necessarily trip o3's chain-of-thought. Majority voting over uncorrelated errors reduces aggregate error below any individual model's rate — the ensemble theorem applied to LLMs.
02
Abstention Recovery
Debate forces resolution on uncertain cases
o3 abstains on 27.2% of questions. In Round 2, an abstaining agent that sees two confident peers with well-reasoned answers re-engages and converges to the correct answer at a high rate. The council recovers these abstentions without lowering any agent's quality bar.
03
Position Change Dynamics
Agents update on evidence, not conformity
Round 2 position changes in the terminal logs are directionally constructive — agents switch wrong→right far more often than right→wrong. Agents don't just copy the majority: they produce revised reasoning that explicitly engages with the peer argument they were exposed to.
04
Confidence Calibration
Vote tallies are better uncertainty signals
A 3-0 unanimous verdict is stronger evidence than any single softmax probability. A 2-1 split signals genuine ambiguity. 93.9% of verdicts were unanimous — when the council agrees, it is almost always right. When it splits, that split itself is informative.

09 · Next Steps

Two Questions Remain

01
Variance Analysis
Is 81.9% stable?
Run the same n = 500 dataset k = 5 times under identical conditions. Compute mean accuracy, standard deviation, and a question-level stability map. If σ < 2 percentage points and stability exceeds 90%, the result is confirmed. If not, the pipeline is stochastically unstable and must be hardened first.
02
Adversarial Robustness
Does debate resist MedFuzz?
MedFuzz (Microsoft Research, 2024) injects misleading patient characteristics into MedQA questions — things that fool solo models without changing the correct answer. GPT-4 drops from 90.2% to 85.4% under a single attack round. The council's debate should catch these flips: a resistant agent in Round 2 provides an anchor the tricked agent must argue against.
CORE CLAIM
If both studies return positive results, the project establishes that structured inter-agent disagreement is a robustness mechanism — not just a coverage mechanism.
